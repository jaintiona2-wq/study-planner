# Study Planner App

A web app to manage study tasks efficiently.

## Features
- Add and delete tasks
- Mark tasks as complete
- Save tasks (local storage)
- Dark mode
- Deadlines
-  Pomodoro timer for focused study sessions

## Built with
- HTML
- CSS
- JavaScript

- ## 💡 Purpose
This project was built to improve productivity and help students stay organized by managing study tasks in a simple and user-friendly interface.
